S3 Private Bucket

ไฟล์เว็บเก็บใน S3 แบบ private ทั้งหมด ไม่เปิด public เข้าถึงได้เฉพาะผ่าน CloudFront เท่านั้น

🌐

CloudFront CDN

กระจายไฟล์ผ่าน edge ทั่วโลก + HTTPS ฟรีจาก AWS Certificate ทำให้โหลดเร็วทุกที่

🔒

Origin Access Control

ใช้ OAC ล็อกให้เฉพาะ CloudFront อ่าน bucket ได้ ปลอดภัยกว่าเปิด S3 website endpoint ตรงๆ

Reusable Deploy

มี script deploy ซ้ำได้ — แก้ไฟล์แล้วรันคำสั่งเดียว sync + invalidate cache ให้อัตโนมัติ
